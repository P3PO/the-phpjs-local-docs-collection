Como ajudar a melhorar a documentação

Há várias maneiras de ajudar a melhorar essa documentação.

Se você encontrar erros no manual, em qualquer língua, notifique-nos utilizando o sistema de bugs em » http://bugs.php.net/. Classifique o problema como "Documentation Problem". Todo problema relacionado a documentação, inclusive sobre os formatos do manual, podem ser reportados como bug.

Nota:

Por favor, não abuse do rastreamento de bugs para solicitar ajuda. Ao invés disso, use uma das muitas » opções de suporte.

Você pode também contribuir com comentários, fornecendo exemplos adicionais e esclarecimentos para outros leitores. Mas não envie notificações de bugs usando o sistema de comentários. Para detalhes, você pode ler mais sobre os comentários na seção 'Sobre os comentários de usuários'.

O manual do PHP é traduzido para muitas linguagens. Se você sabe Inglês e alguma outra língua, você pode ajudar com as traduções. Se você deseja iniciar uma nova tradução, ou ajudar num projeto de tradução, leia o » https://wiki.php.net/doc/howto.

O Projeto de Documentação do PHP tem um canal de IRC onde vários autores aparecem. No #php.doc na irc.efnet.org e discutir sobre melhorias para a documentação.
