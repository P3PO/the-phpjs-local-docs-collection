Como nós geramos os formatos

Este manual é escrito em XML, utilizando o » DocBook XML DTD e usando » PhD (The [PH]P based [D]ocBook renderer) para manutenção e formatação.

Utilizando XML como meta dado nos permite gerar vários outros formatos a partir dos arquivos fonte, ao mesmo tempo que a manutenção é feita em um único lugar para todas as versões. As ferramentas utilizadas para formatar o manual online são » PhD. Usamos » Microsoft HTML Help Workshop para criar o formato HTML Help e, é claro, o próprio PHP para fazer algumas conversões adicionais.

O PHP é gerado em várias linguagens e formatos, veja » http://www.php.net/docs.php para mais detalhes. O código-fonte do XML pode ser obtidos do CVS e visualiados em » http://cvs.php.net/. A documentação está armazenada no módulo phpdoc.

Para maiores informações sobre como obter os fontes XML desta documentação, veja em » http://cvs.php.net/. A documentação é guardada no módulo phpdoc.
