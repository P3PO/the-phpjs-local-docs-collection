Introdução

Essas funções permitem você acessar Banco de Dados Oracle 10, Oracle 9, Oracle8 e Oracle7 usando a Oracle Call Interface (OCI). Elas suportam ligação de variáveis PHP para Oracle placeholders, tem suporte completo a LOB, FILE e ROWID, ae permite você definir variáveis.
