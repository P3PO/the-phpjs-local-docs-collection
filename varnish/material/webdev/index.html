The following files are provided for the Varnish training material:

  • ajax.html - Used to test maqued and unmasqued AJAX calls
  • article.php - Basic testing of cache control headers.
  • cookies.php - Displays all cookies the server sees.
  • httpheadersexample.php - Test various cache control headers, both for varnish and the browser.
  • purgearticle.php - Demonstration of how an application can do a “callback” to Varnish to purge content.
  • esi-top.php - Top page for ESI-testing
  • esi-user.php - User-specific content for ESI
  • test.php - Test custom cache-control values
  • set-cookie.php - Set a cookie!
