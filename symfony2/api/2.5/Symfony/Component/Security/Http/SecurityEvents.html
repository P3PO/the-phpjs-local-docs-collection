Constants

INTERACTIVE_LOGIN

The INTERACTIVE_LOGIN event occurs after a user is logged in interactively for authentication based on http, cookies or X509.

The event listener method receives a Symfony\Component\Security\Http\Event\InteractiveLoginEvent instance.
SWITCH_USER

The SWITCH_USER event occurs before switch to another user and before exit from an already switched user.

The event listener method receives a Symfony\Component\Security\Http\Event\SwitchUserEvent instance.
