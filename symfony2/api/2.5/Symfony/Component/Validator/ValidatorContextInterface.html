Symfony\Component\Validator\ValidatorContextInterface

interface ValidatorContextInterface

Stores settings for creating a new validator and creates validators

The methods in this class are chainable, i.e. they return the context object itself. When you have finished configuring the new validator, call getValidator() to create the it.

$validator = $context ->setClassMetadataFactory($customFactory) ->getValidator();
